Inline, Inline-block or block Elements

Every HTML element has a default display value, depending on what type of element it is. There are three display values: block inline and Inline-block.

Block Elements

A block-level element is an HTML element that begins a new line on a web page and extends the full width of the available horizontal space of its parent element. It creates large blocks of content like paragraphs or page divisions. Block-level elements are used within the HTML document's body.

Semantic Tag

Semantic elements = elements with a meaning. A semantic element clearly describes its meaning to both the browser and the developer.

Examples of non-semantic elements: div and span - Tells nothing about its content.

Examples of semantic elements: form, table, and article - Clearly defines its content.
